### 𐩒 MANUAL OPÉRATIONNEL (EXTRACT)

#### ⌬ EXÉCUTION DYNAMIQUE "THE LIFE-CODE"
Le moteur interprète vos saisies pour modifier le monde en temps réel via des injections CSS/SVG physiques.

*   **`mutate()`** : Corruption des entités IA adverses via signature de pixel.
*   **`spawn(Guardian)`** : Génération de boucliers physiques autour de l'ID utilisateur.
*   **`while(true)`** : Instanciation d'un **BLACK HOLE** (Singularité de purge).
*   **`svg(dragon)`** : Injection de vecteurs animés complexes dans la topologie de la grille.

#### ⌬ PHYSIQUE DU STYLE (CSS INJECTION)
| Commande | Effet Systémique |
| :--- | :--- |
| `color: #hex` | Réécriture chromatique de la zone d'influence. |
| `opacity: 0` | Mode furtif (Perte de traçabilité IA). |
| `font-weight: bold` | Solidification atomique des murs de code. |
| `CSS.animate()` | Multiplication de la puissance (`PWR`) du curseur. |
